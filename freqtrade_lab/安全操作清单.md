# Freqtrade Lab 安全操作清单

## 每次打开 WebUI 前检查

- 是否通过 SSH tunnel 访问。
- 是否没有开放 `8080` 公网端口。
- 是否没有上传 `.env`。
- 是否没有配置真实 API key。
- 是否没有运行 Freqtrade 的运行机器人命令。
- 是否 `dry_run=true`。
- 是否没有 live 配置。
- 是否没有 futures/leverage 实盘配置。
- 是否没有 Telegram token。
- 是否没有真实账户余额显示。
- 是否只运行 `backtest`、`download-data`、`show-config`、`version`。

## 红色禁止项

- 禁止接入真实 API key。
- 禁止点击或运行 live trade。
- 禁止运行带真实 API 的 paper/dry-run。
- 禁止开启 futures live。
- 禁止杠杆。
- 禁止两个 bot 同账户跑资金费套利。
- 禁止把 WebUI 暴露公网。
- 禁止把 backtest 结果当成盈利承诺。

## 出现异常时

如果你看到真实账户余额、真实持仓、真实资金流水、API key 输入项已填充、live 状态、futures live 状态、leverage 状态，立即停止 WebUI，并不要继续操作。

当前项目允许的是研究、回测、公开数据下载和配置校验，不允许交易。
