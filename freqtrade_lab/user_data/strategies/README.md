# Freqtrade Strategies

This is the only directory for new single-leg strategy research.

- `M1ATrendValidationStrategy.py` is a frozen historical failed-validation reproduction.
- `M1BFuturesFundingProbeStrategy.py` is a futures-leg capability probe, not a two-leg arbitrage strategy.
- `BTCETHRelativeStrengthRotation.py` is the fixed M1C public-data research candidate. It is not yet performance validated.
- No strategy in this directory is approved for M2, paper trading with real API credentials, or live trading.
- Hyperparameter rescue of failed validations is prohibited.
