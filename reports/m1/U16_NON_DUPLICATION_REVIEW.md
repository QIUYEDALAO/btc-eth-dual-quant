# U-16 Non-Duplication Review

- Status: `pass_design_level`

U-16 requires a change in the relationship between an asset's completed
multi-observation return path and the contemporaneous peer common path, jointly
with positive relative displacement. It is not U-04 single-period negative
residual reversal, U-07 stress-state relative strength, U-09 low residual
volatility, U-10 price/volume trend confirmation, U-11 conditional beta capture
or U-13 leader-to-laggard diffusion. It does not use U-15 taker fields.

The mechanism can fail through noisy correlation estimates, terminal spikes,
common dispersion regimes, nonstationarity, lifecycle changes or costs. This
review establishes design-level independence only; it provides no evidence of
feasibility or profitability.
